## Описание проекта
Это проект экрана приложения, написанный на Swift c использованием SwiftUI. На экране присутствуют следующие элементы:

- **Заголовок**: Приветствие пользователя.
- **Информация о докторе**: Информация о выбранном докторе.
- **Поиск**: Input Field с поиском.
- **Кнопки информации**: Кнопки для просмотра различной информации.
- **Доктора рядом**: Информация о докторах рядом.
- **Кнопки навигации**: Кнопки для навигации по приложению.

## Структура проекта
- **`Digital_Department_IOSApp`**: Главный файл для компиляции всего приложения.
- **`Header`**:  Приветствие пользователя.
- **`DoctorInfo`**: Информация о выбранном докторе.
- **`Search`**: Input Field с поиском.
- **`InfoButtons`**: Кнопки для просмотра различной информации.
- **`NearDoctors`**: Информация о докторах рядом.
- **`NavigationButtons`**: Кнопки для навигации по приложению.

## Технологии
- SwiftUI
